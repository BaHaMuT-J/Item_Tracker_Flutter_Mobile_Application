# Item Tracker

Simple item tracker mobile app built with Flutter. User can create new categories, and items in each category to keep track of item they have or want.

## Features

- Add new category
- Rename category
- Delete category
- Reorder category
- Add new item
- Delete item
- Reorder item
- Shared preferences to keep data across session
